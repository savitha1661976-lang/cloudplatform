Cloud Platform

Cloud platform refers to internet-based remote servers and operating systems. It provides scalable computing resources such as servers, storage, databases, analytics, networking, software and AI. Organizations avoid maintaining physical data centers and instead use flexible cloud infrastructure.

Cloud Service Providers

  • Software as a Service[SaaS], Platform as a service[PaaS], Infrastructure as a service[IaaS], network services, business applications, mobile applications
  • Cloud Engineers | Devops Engineers
  • The cloud service providers host these services in a data center, and users can access these services through cloud provider companies using an Internet connection.

Amazon Web Services (AWS)

  • Scalable infrastructure | Flexible storage
  • Pay-as-you-go pricing- Mainly used for the companies letting more offers. increased storage
  • Flexible storage- Amazon S3, EBS, EFS
  • Data encryption & DDoS protection
  • Identity & access management

Microsoft Azure

  • IaaS infrastructure allows us to launch a general-purpose virtual machine in different platforms such as Windows and Linux.
  • Scalable & flexible
  • Azure provides tools to: Create websites, Deploy applications, Monitor performance, Fix errors quickly
  • Azure is very powerful when: A company has local servers but also wants to use cloud services
  • Microsoft Azure is best for AI and Machine Learning as it offers AI tools, chatbots, vision recognition, language processing, and Azure OpenAI services, making it ideal for AI projects, data science, and automation systems.
  • It managed each resource individually
  • Microsoft Azure is best for companies already using Windows Server, SQL Server, .NET applications, Office 365, and SharePoint because it integrates and works smoothly with Microsoft products

Google Cloud Platform

  • BigQuery (data warehouse used to analyze huge datasets very fast), Dataproc (processing big data | data scientists), Pub/Sub (messaging service | Sends data on real time)
  • Virtual Private Cloud - You can create your own virtual network | Control IP addresses, firewalls, routing
  • Serverless services - Google automatically handles scaling | You only pay when your code runs
  • High-performance networking - Fast global network | Low latency | Secure connections | Load balancing

IBM Cloud

  • AI-powered data tools
  • IaaS, PaaS, SaaS
  • Operational efficiency- Less manual work(Automated) | Lower cost | Better performance | Faster deployment | Redcued downtime

VMware Cloud

  • Hybrid architecture- On-premises + public cloud
  • Subscription model - pay as you use & montly subscription
  • SDDC(Software Defined Data Center) clusters- Everything is created and managed using software, not physical hardware manually
  • Flexible storage | as application basis
  • provides a dedicated high-performance network for managing the application traffic and also supports multicast networking
  • eliminates the time and cost complexity

Oracle Cloud

  • Supports Java, Ruby, PHP, Node.js
  • DevOps integration- Tools like docker
  • Oracle Autonomous Database is a cloud database that uses AI and machine learning to automatically manage, secure, and repair itself—reducing the need for manual DBA work.
  • It offers customizable Virtual Cloud Networks, firewalls, and IP addresses to securely support private networks

Aneka

  • Aneka is a Platform-as-a-Service (PaaS) designed mainly for building and managing distributed and cloud applications using .NET and other technologies
  • Unlike many cloud platforms, Aneka supports multiple programming models in one platform: Task Model – For independent tasks, Thread Model – For parallel programming, MapReduce Model – For big data processing
  • can integrate Private, Public & Hybrid models

Force.com & Salesforce.com

  • Force.com(Paas) – Platform to build apps | Used by developers
  • Developers write code using Apex (a language similar to Java) | These apps run on Salesforce - Force.com
  • Salesforce(Saas) – Ready-made CRM software | Used by companies and users
  • Businesses use it to manage customers, sales, and services inside salesforce - Salesforce
  • AppExchange(buy,sell apps platform) & Trailhead(learning platform Salesforce)
